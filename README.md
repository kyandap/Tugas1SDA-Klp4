## Daftar Program

1. Menggabungkan beberapa linked list menjadi satu (descending)
2. Partisi list berdasarkan nilai tertentu
3. Menghapus duplikasi data
4. Menghapus node ke-n dari akhir
5. Ganjil Genap List
6. Cek apakah list palindrom
7. Menghapus elemen tertentu
8. Reorder list
9. Reverse sebagian list
10. Menghapus node berdasarkan array

## Cara Menjalankan Program

1. Compile:

   ```
   gcc main.c -o program
   ```
2. Jalankan:

   ```
   ./program
   ```

## Output

Program menampilkan hasil sesuai masing-masing soal, seperti:

* Linked list hasil modifikasi
* Pengelompokan ganjil/genap
* Status palindrom (true/false)
